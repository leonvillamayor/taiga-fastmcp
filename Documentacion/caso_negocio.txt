Queremos generar un servidor mcp utilizando el framework fastmcp (https://github.com/jlowin/fastmcp) que sea capaz de realizar cualquier tipo de operacion sobre taiga. Para lograrlo:

Utilizando la documentación recogida en Documentacion/fastmcp.md y Documentacion/taiga.md:
- Crearemos un servidor mcp, utilizando todas las capacidades necesarias de fastmcp y usandolas de forma efectiva y buenas practicas, que permita los protocolos de transporte (ambos y por separado): stdio y HTTP Transport (Streamable) que ofrezca herramientas
para interactuar con taiga, utilizando el documento Documentacion/taiga.md donde se explican cada una de sus funcionalidades y que deberan de ser codificadas en forma de herramientas
que se expondran en el servidor mcp.
- Se verificará que el servidor mcp ha aplicado correctamente las buenas practicas y funcionalidades de fastmcp y que expone todas las herramientas para trabajar con taiga. Entre otras, el servidor mcp debe
de exponer y orquestar todas las funcionalidades recogidas en Documentacion/taiga.md mediante herramientas del servidor mcp.
